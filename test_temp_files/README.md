# Планировщик изолирования и скрутки жил кабеля

## Контекст

Инструмент оперативного планирования производства на кабельном заводе.
Задача: распределить жилы нескольких кабелей из заказа по имеющимся барабанам ТПЖ (токопроводящая жила), сформировать инструкцию на изолирование и скрутку.

## Архитектура алгоритма

### Структуры данных

```python
@dataclass
class Drum:
    id: str       # уникальный ID барабана, напр. 'Барабан-1'
    length: int   # длина провода в метрах

@dataclass
class Cable:
    name: str           # марка / позиция заказа
    min_length: int     # минимальная длина жилы (м)
    cores: int          # число жил
    colors: List[str]   # идентификаторы жил: цвета ('ж/з','синий',...) или пары ('П1','П2',...)
    flexible: bool      # True → алгоритм может удлинить жилу для поглощения остатков

@dataclass
class Allocation:
    cable_name: str   # к какому кабелю
    color: str        # жила/пара
    drum_id: str      # с какого барабана ТПЖ
    length: int       # сколько метров
```

### Этапы решения (`solve()`)

```
1. OR-Tools CP-SAT
   ┌─────────────────────────────────────────────────────────┐
   │ Переменная x[кабель, барабан] = метров с этого барабана │
   │ Одинакова для ВСЕХ жил кабеля (физически корректно)     │
   │                                                         │
   │ Ограничения:                                            │
   │   Σ x[k,d] × жил[k] ≤ ёмкость барабана d               │
   │   Σ x[k,d] ≥ min_length[k]  (или =, если не гибкий)    │
   │   ненулевой сегмент ≥ MIN_SEGMENT                       │
   │   MAX_SPLITS=1 → жила одним куском (без стыков)         │
   │                                                         │
   │ Цель: max(WASTE_WEIGHT × использованный провод          │
   │            − число сегментов)                           │
   └─────────────────────────────────────────────────────────┘

2. Сортировка порядка цветов (_best_color_order)
   • ≤8 цветов → перебор всех перестановок (точно)
   • >8 цветов → жадный алгоритм ближайшего соседа
     (нужно для парных кабелей 8×2×1, 16×2×1 с 8–16 парами)

3. Группировка кабелей по составу жил (_group_cables_by_colors)
   • Кабели с одинаковым frozenset(colors) → одна группа
   • Метки групп: '5ж', '4ж', '3ж', '2ж', 'Парные'
   • Если две группы с одним числом жил → суффиксы '4ж-А', '4ж-Б'
   • КРИТИЧНО: FFD запускается РАЗДЕЛЬНО для каждой группы

4. FFD пакование в наборы для скрутки (_pack_output_drums_by_group)
   • First-Fit Decreasing по каждому цвету внутри группы
   • Набор (Нi) = один мотки по каждому цвету, скручиваемые вместе
   • Метки: '5ж-Н1-ж/з', '5ж-Н1-синий', '4ж-Н1-ж/з', ...
   • Порядок жил на барабане = порядок инструкции изолирования

5. Инструкция изолировщика (create_instructions)
   • Последовательность: цвет за цветом, кабель за кабелем
   • Отмечает смены барабана ТПЖ и смены цвета

6. Инструкция скрутчика (create_twisting_instruction)
   • ОБРАТНЫЙ порядок сегментов на барабане:
     изолировщик мотает снизу вверх →
     скрутчик разматывает сверху вниз

7. Таблица приёмных барабанов (create_receiving_drums_table)
   • Показывает загрузку каждого мотка после изолировки
```

### Почему FFD по группам, а не общий?

Смешанный заказ (5-жильные + 4-жильные кабели): у 4-жильных нет жилы 'синий'.
Если запустить один FFD, число мотков по 'синий' = 2, по остальным = 3 → набор Н3
не может быть скручен (нет синего мотка). Решение: группы, раздельный FFD.

## Парные кабели (ТПП, КВП и др.)

Пример: `ТПП 4×2×1` = 4 пары × 2 жилы × 1 мм²

**Модель в алгоритме:**
- `cores = 8` (физическое число жил, для расхода ТПЖ)
- `colors = ['П1', 'П2', 'П3', 'П4']` — идентификаторы пар
- Одна пара = 2 абсолютно одинаковые жилы → оба провода берутся с одного барабана, одной длиной

**Скрутка парных кабелей — двухэтапная (НЕ реализовано, план v5+):**
```
Этап 1: попарная скрутка (П1: жила-а + жила-б → витая пара 1)
Этап 2: кабельная скрутка (П1 + П2 + П3 + П4 → кабель)
```

## Глоссарий

| Термин | Значение |
|--------|---------|
| ТПЖ | Токопроводящая жила — неизолированный провод на входном барабане |
| Жила | Изолированный провод после изолировочной машины |
| Набор (Нi) | Комплект мотков по всем цветам для одного прохода скрутки |
| Мотка / приёмный барабан | Барабан с изолированным проводом (выход изолировки, вход скрутки) |
| Прогон | Один непрерывный цикл изолировочной машины |
| Строительная длина | Длина одного отрезка кабеля из кабельного журнала |
| FFD | First-Fit Decreasing — алгоритм пакования отрезков в наборы |
| Гибкий кабель | Позиция заказа, длина которой может быть увеличена для поглощения остатков ТПЖ |
